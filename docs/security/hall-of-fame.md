# Security Hall of Fame

Listed with their consent.

| _none yet_ |            |                                          |          |

## Adding a Researcher

Add a row when:

1. The finding has been triaged and confirmed
2. The researcher has provided written consent to be listed
3. The fix has been merged (or an exception has been documented)

## Anonymity

Researchers may request partial anonymization (e.g., "handle: foo"). The
full name is never published without explicit consent.
