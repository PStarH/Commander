    Class JsonMemoryStore

    JSON-file backed MemoryStore for simple persistence. Falls back gracefully when SQLite is unavailable.

    Uses BM25 scoring (Okapi BM25) for high-quality full-text search, matching the search quality of SQLite FTS5.
