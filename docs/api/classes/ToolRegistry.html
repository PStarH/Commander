    Class ToolRegistry

    ToolRegistry — Auto-Discovery Tool Registry

    Inspired by Hermes Agent's tools/registry.py (auto-registration at import time) and OpenClaw's tool registration pattern.

    Tools self-register via ToolRegistry.register() during initialization. This replaces the manual createAllTools() approach with a cleaner, extensible pattern.

    Usage: // In any tool file: ToolRegistry.register(new MyTool());

    // In runtime setup: const allTools = ToolRegistry.getAllTools();
