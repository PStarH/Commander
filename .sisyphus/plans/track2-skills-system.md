# Track 2: Skills System — Detailed Implementation Plan

**Status**: Draft  
**Estimated**: 5-7 days  
**Dependencies**: MetaLearner (existing), ThreeLayerMemory (existing), Orchestrator (existing)

---

## Overview

Commander already has a minimal skills system at `packages/core/src/skills/index.ts`:
- `SkillDef` interface with name, description, tools, prompt, usageCount
- JSON file storage in `.commander/skills/`
- `createSkill`, `loadSkill`, `listSkills`, `buildSkillsPrompt`, `recordSkillUsage`, `deleteSkill`
- Level 0/1 progressive disclosure support

The gap closure plan calls for a substantially enhanced system that:
1. Follows **agentskills.io SKILL.md format** (YAML frontmatter + markdown body) for ecosystem compatibility
2. **Bridges MetaLearner** — Thompson Sampling automatically generates skills when confidence > 0.7 on ≥5 trials
3. **Integrates with ThreeLayerMemory** procedural layer as a storage backend option
4. Has a **curator/pruning mechanism** (like Hermes Curator — weekly cycle, grades skills, consolidates, prunes)
5. **Agent-visible skills** injected into system prompt at execution start with progressive disclosure
6. **Community skill hub** interface (reserved for future)

`suggestForTask` algorithm:
1. Extract keywords from goal (remove stop words, keep nouns + verbs)
2. Score each skill by:
   - Keyword overlap (0-1)
   - Category match bonus (+0.2)
   - Quality score multiplier
   - Usage count boost (logarithmic)
3. Return top N by score

### Step 5: MetaLearner Bridge (`metaLearnerBridge.ts`)

**File**: `packages/core/src/skills/metaLearnerBridge.ts`

**The core innovation**: Thompson Sampling auto-generates skills when a strategy proves consistently effective.

```typescript
export class MetaLearnerBridge {
  private metaLearner: MetaLearner;
  private skillManager: SkillManager;

  constructor(metaLearner: MetaLearner, skillManager: SkillManager) { ... }

  /**
   * Extract skills from Thompson Sampling priors.
   * Called after each execution (non-critical, best-effort).
   */
  async extractSkills(): Promise<Skill[]> {
    const newSkills: Skill[] = [];

    for (const [taskType, priors] of this.metaLearner.getThompsonPriors()) {
      const bestIdx = priors.map(p => p.mean).indexOf(Math.max(...priors.map(p => p.mean)));
      const bestScore = priors[bestIdx].mean;
      const totalTrials = priors[bestIdx].totalTrials;

      // Threshold: ≥5 trials and >0.7 confidence
      if (totalTrials >= 5 && bestScore > 0.7) {
        const strategyName = ['SEQUENTIAL', 'PARALLEL', 'HANDOFF', 'MAGENTIC', 'CONSENSUS'][bestIdx];
        const skillName = `strategy-${taskType}-${strategyName}`.toLowerCase();

        // Check if already exists
        const existing = await this.skillManager.search({ text: skillName });
        if (existing.length > 0) continue;

        const skill = await this.skillManager.create(
          skillName,
          this.generateSkillContent(taskType, strategyName, bestScore),
          {
            category: 'strategy',
            tags: [taskType, strategyName, 'auto-generated'],
            source: 'learned',
            qualityScore: bestScore,
            usageCount: totalTrials,
            avgSuccessRate: bestScore,
            autoGenerated: true,
            generatedFrom: `strategy-${taskType}-${strategyName}`,
            createdAt: new Date().toISOString(),
            updatedAt: new Date().toISOString(),
          },
        );
        newSkills.push(skill);
      }
    }

    return newSkills;
  }

  /**
   * Generate agentskills.io-compatible markdown content.
   */
  private generateSkillContent(taskType: string, strategy: string, confidence: number): string {
    return [
      `# strategy-${taskType}-${strategy}`,
      '',
      '## Description',
      `Automatically discovered strategy for ${taskType} tasks using ${strategy} topology.`,
      `Confidence: ${(confidence * 100).toFixed(0)}% over trials.`,
      '',
      '## When to Use',
      `- Task type matches: ${taskType}`,
      `- The ${strategy} topology has shown consistently positive results`,
      '',
      '## Steps',
      `1. Configure ${strategy} orchestration topology`,
      '2. Decompose task using ROMA recursive atomizer',
      '3. Execute subtasks with ${strategy} coordination pattern',
      '4. Synthesize results using LEAD_SYNTHESIS',
      '',
      '## Tools Required',
      '- Agent delegation (agent tool)',
      '- File system tools for persistence',
      '- Web search for research tasks',
    ].join('\n');
  }
}
```

### Step 6: Skill Injector (`skillInjector.ts`)

**File**: `packages/core/src/skills/skillInjector.ts`

**Responsibilities**:
- At execution start, inject relevant skills into the agent's system prompt
- Progressive disclosure: Level 0 (names only), Level 1 (full content for top 3), Level 2 (file references)

```typescript
export class SkillInjector {
  private manager: SkillManager;

  constructor(manager: SkillManager) { ... }

  /**
   * Build the skills block for system prompt.
   * Called by AgentRuntime.buildSystemPrompt().
   * 
   * Progressive disclosure:
   *   Level 0: all skill names + descriptions (~3K tokens for 100 skills)
   *   Level 1: full content for top-3 most relevant skills
   *   Level 2: reference paths to load on demand
   */
  async buildSkillsBlock(
    goal: string,
    maxLevel: DisclosureLevel = 0,
    tokenBudget?: number,
  ): Promise<string> {
    const catalog = await this.manager.list();
    if (catalog.length === 0) return '';

    // Always include Level 0
    const level0 = [
      '## Available Skills',
      ...catalog.map(s => `- ${s.name}: ${s.description} (category: ${s.category}, quality: ${(s.qualityScore * 100).toFixed(0)}%)`),
    ].join('\n');

    if (maxLevel === 0) return level0;

    // Level 1: full content for top-relevant skills
    const relevant = await this.manager.suggestForTask(goal, 3);
    const level1 = relevant.length > 0
      ? '\n\n### Recommended Skills\n' + (await Promise.all(
          relevant.map(async s => {
            const skill = await this.manager.get(s.name);
            return skill ? `\`\`\`skill\n${skill.content}\n\`\`\`` : '';
          }),
        )).filter(Boolean).join('\n\n')
      : '';

    if (maxLevel === 1) return level0 + level1;

    // Level 2: file references for all skills
    const level2 = '\n\n### Skill References\n' +
      catalog.map(s => `- \`.commander/skills/${s.name}.md\``).join('\n');

    return level0 + level1 + level2;
  }
}
```

### Step 7: Skill Curator (`skillCurator.ts`)

**File**: `packages/core/src/skills/skillCurator.ts`

**Hermes-inspired curator**: periodic cycle that grades skills, consolidates duplicates, prunes low-quality.

```typescript
export class SkillCurator {
  private manager: SkillManager;
  private readonly QUALITY_THRESHOLD = 0.3;
  private readonly MIN_USAGE_THRESHOLD = 2;
  private readonly MAX_SKILLS = 200;

  constructor(manager: SkillManager) { ... }

  /**
   * Run full curation cycle.
   * Called periodically (every N executions or weekly).
   */
  async curate(): Promise<CuratorReport> {
    const report: CuratorReport = {
      pruned: [],
      consolidated: [],
      qualityDropped: [],
      totalBefore: 0,
      totalAfter: 0,
    };

    const catalog = await this.manager.list();
    report.totalBefore = catalog.length;

    // 1. Prune low-quality skills
    for (const entry of catalog) {
      if (entry.qualityScore < this.QUALITY_THRESHOLD && entry.usageCount < this.MIN_USAGE_THRESHOLD) {
        await this.manager.delete(entry.name);
        report.pruned.push(entry.name);
      }
    }

    // 2. Consolidate duplicates (same name, similar content)
    // Simple heuristic: same category + overlapping tags
    const grouped = this.groupSimilar(catalog);
    for (const group of grouped) {
      if (group.length > 1) {
        // Keep the one with highest quality, delete rest
        group.sort((a, b) => b.qualityScore - a.qualityScore);
        for (const dup of group.slice(1)) {
          await this.manager.delete(dup.name);
          report.consolidated.push(dup.name);
        }
      }
    }

    // 3. Enforce max skills cap
    const finalCatalog = await this.manager.list();
    report.totalAfter = finalCatalog.length;
    if (finalCatalog.length > this.MAX_SKILLS) {
      const toPrune = finalCatalog
        .sort((a, b) => a.usageCount - b.usageCount)
        .slice(0, finalCatalog.length - this.MAX_SKILLS);
      for (const entry of toPrune) {
        await this.manager.delete(entry.name);
        report.pruned.push(entry.name);
      }
      report.totalAfter = this.MAX_SKILLS;
    }

    return report;
  }

  private groupSimilar(entries: SkillCatalogEntry[]): SkillCatalogEntry[][] {
    // Group by normalized name or overlapping tags
    const groups = new Map<string, SkillCatalogEntry[]>();
    for (const entry of entries) {
      // Normalize: lowercase, remove non-alphanumeric
      const key = entry.tags.sort().join(',');
      const existing = groups.get(key) || [];
      existing.push(entry);
      groups.set(key, existing);
    }
    return Array.from(groups.values()).filter(g => g.length > 1);
  }
}

export interface CuratorReport {
  pruned: string[];
  consolidated: string[];
  qualityDropped: string[];
  totalBefore: number;
  totalAfter: number;
}
```

**Trigger points**:
1. After every `extractSkills()` call in orchestrator post-execution (lightweight)
2. CLI command: `commander skill curate` (full cycle)
3. Automatic: via CRON-like schedule (future)

### Step 9: Global Factory

/**
 * Create the full skills system with all components wired together.
 */
export function createSkillSystem(memory?: ThreeLayerMemory): {
  manager: SkillManager;
  injector: SkillInjector;
  curator: SkillCurator;
  bridge: MetaLearnerBridge;
} {
  const store = new SkillStore(undefined, memory);
  const manager = new SkillManager(store);
  const injector = new SkillInjector(manager);
  const curator = new SkillCurator(manager);
  const bridge = new MetaLearnerBridge(getMetaLearner(), manager);
  return { manager, injector, curator, bridge };
}

// Singleton (lazy)
let globalSkillSystem: ReturnType<typeof createSkillSystem> | null = null;

export function getSkillSystem(): ReturnType<typeof createSkillSystem> {
  if (!globalSkillSystem) {
    globalSkillSystem = createSkillSystem(getGlobalThreeLayerMemory());
  }
  return globalSkillSystem;
}

export function resetSkillSystem(): void {
  globalSkillSystem = null;
}

## Risk Mitigation

| Risk | Mitigation |
|------|------------|
| YAML parsing dependency | Use `js-yaml` (widely used, 10M+ weekly downloads). If missing, graceful fallback to JSON format |
| Performance overhead from skill injection | Level 0 only (~3K tokens for 100 skills). Level 1 only for top 3 relevant skills |
| Disk I/O from skill file operations | In-memory LRU cache in SkillManager. Writes are async and batched |
| Too many auto-generated skills | Quality threshold (0.7), curator prunes below 0.3 quality and <2 usages. Max 200 skills cap |
| Breaking existing API | Backward-compatible wrappers keep old `createSkill()`, `loadSkill()`, etc. working |
